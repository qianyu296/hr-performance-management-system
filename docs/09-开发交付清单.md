## 2. 代码与迁移交付结构

```text
backend/
  src/main/java/com/hrpm/{common,security,system,organization,personnel,workflow,
                           attendance,performance,notification,report,file}/
  src/main/resources/db/migration/
    V001__system_and_organization.sql
    V002__workflow.sql
    V003__attendance.sql
    V004__goals_and_performance.sql
    V005__file_async_audit_report.sql
    V006__seed_r1.sql
  src/test/
frontend/
  src/{api,views,components,stores,router,composables,types,utils}/
docs/
  hrpm-v1.openapi.yaml
```

一个 Flyway 迁移只做一个可审查的数据库变更，不修改已经发布的迁移文件。每份迁移必须同时提供：DDL、必要的数据回填、迁移后校验 SQL、影响说明。R2 用新迁移扩展 R1，不得清空或重建生产业务表。

## 3. 实现顺序

1. 建立工程骨架、统一异常、认证会话、审计上下文、Flyway、Testcontainers 和 OpenAPI 校验；完成 `V001` 与 R1 种子数据。
2. 实现组织、员工、用户角色、数据范围和字段脱敏。先完成权限 Repository 组件，再开发任何业务查询。
3. 实现工作流模板、实例、任务、动作日志和待办，并以请假提交/审批作为第一条端到端流程。
4. 实现假期规则、工作日历、余额流水和并发控制；完成请假后再实现绩效配置、任务快照、评分与发布。
5. 实现文件隔离上传、outbox、异步任务、导出与报表快照。通知、导出和批量发布必须复用同一作业框架。
6. R1 验收后实现人事异动完整流程、离职交接、加班调休、目标树、校准申诉、结果版本及 R2 报表。

每步完成前必须包含对应 Controller、Application Service、Repository、DTO、OpenAPI、前端页面、单元/集成/API/E2E 测试。禁止只创建表或只创建页面而没有状态动作和权限校验。

## 4. 环境配置清单

| 变量 | 示例用途 | 规则 |
| --- | --- | --- |
| `DB_URL`、`DB_USERNAME`、`DB_PASSWORD` | MySQL 连接 | 生产使用独立最小权限账号 |
| `REDIS_URL`、`REDIS_PASSWORD` | 会话和缓存 | Redis 不保存唯一业务事实 |
| `MINIO_ENDPOINT`、`MINIO_ACCESS_KEY`、`MINIO_SECRET_KEY`、`MINIO_BUCKET` | 私有附件桶 | 仅服务端持有密钥 |
| `JWT_SIGNING_KEY`、`DATA_ENCRYPTION_KEY` | 令牌和敏感字段加密 | 生产从密钥管理服务注入并支持轮换 |
| `APP_TIME_ZONE` | 企业展示时区 | 数据库存储 UTC，默认 `Asia/Shanghai` |
| `FILE_MAX_SIZE`、`FILE_ALLOWED_TYPES` | 上传策略 | 默认最大 50 MB，白名单媒体类型 |
| `ASYNC_MAX_RETRY`、`ASYNC_LOCK_SECONDS` | 作业恢复 | 默认重试 5 次，锁定 300 秒 |

开发环境通过 Docker Compose 提供 MySQL、Redis 和 MinIO；测试环境使用隔离实例或 Testcontainers；生产环境环境变量由部署平台注入。所有环境必须执行同一组 Flyway 迁移。

## 5. Definition of Done

一个功能完成必须同时满足：

1. 需求编号、数据表、OpenAPI operation、前端路由和测试用例可追溯。
2. 所有查询、下载、导出和动作在服务端执行功能、字段和数据范围校验。
3. 状态变更只走应用服务，具有幂等、乐观锁和审计；需要异步处理的行为使用 outbox 和作业表。
4. 迁移在空库和含历史测试数据的库均可执行，唯一约束、索引和数据回填经过验证。
5. 自动化测试满足 `07` 的质量门禁，前端处理加载、空态、无权、冲突和网络错误。
6. 变更更新相应文档、OpenAPI 和测试用例；未同步文档的实现不得合并。
